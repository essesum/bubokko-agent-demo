# БубOkko — кейс видео-агента

Презентационный сайт: готовый бренд-ролик персонажа **БубOkko**, созданный нашим ИИ-видео-агентом с авторским режиссёрским подходом.

**Логика:** сначала показываем результат (видео) → затем объясняем, что это сделал агент, и как устроен режиссёрский подход внутри него.

## Что внутри
- `index.html` — слайд-дек лендинг (результат → раскадровка → reveal «это сделал агент» → подход → пайплайн → доказательства → CTA).
- `workbook.html` — полный интерактивный режиссёрский воркбук (Director's Treatment & Production Pack).
- `assets/bubokko_result.mp4` — итоговый ролик (с клонированным голосом).
- `assets/frames/` — реальные рендеры персонажа из бренд-библии.

## Режиссёрский подход агента
Rule of Six (Murch) · Character Lock (positive + negative) · покадровый shot list с таймкодами · промпт-движок Seedance 2.0 / Kling · клон голоса · QA-гейт.
